# よく使う依頼文スニペット集

Claude（Web版・Claude Code）で頻繁に使う短い依頼文のパターン集。
コピーしてそのまま使うか、必要に応じて書き換えて使う。

---

## コードレビュー

```
以下のコードをレビューしてください。
バグ・セキュリティ上の問題・改善点があれば指摘してください。
ただし、動いている部分のリファクタリングは不要です。問題点のみ教えてください。
```

---

## バグ調査

```
次のエラーが発生しています。原因と修正方法を教えてください。

【エラーメッセージ】


【再現手順】


【期待する動作】

```

---

## 実装依頼（過剰実装を防ぐ）

```
以下の機能を実装してください。

【要件】


【制約】
- 既存コードのスタイルに合わせる
- 不要な抽象化・ヘルパー関数は作らない
- 今回の要件に関係ないコードは変更しない
```

---

## ドキュメント作成

```
以下のコードの説明を書いてください。
対象読者は[エンジニア / 非エンジニア]です。
技術的な詳細より、何をするものかと使い方を中心に書いてください。
```

---

## セッション開始（Claude Code 向け）

```
docs/handoff.md を読んで現在の状態を把握してください。
把握できたら「現在地の認識」を簡潔にまとめて、今日やることを確認させてください。
```

---

## セッション終了（Claude Code 向け）

```
作業を終了します。以下を実施してください。

1. typecheck && lint を実行してエラーがないか確認
2. 問題なければ今日の変更をコミット
3. docs/session-log.md の先頭に今日の作業記録を追記
4. docs/handoff.md を現在の状態に更新
```

---

## 調査・選択肢の提示

```
[テーマ] について調べています。
選択肢と各トレードオフを簡潔にまとめてください。
結論を出すのはこちらで行うので、推薦はしなくて構いません。
```

---

```
以下のコードが何をしているか説明してください。
難しい部分・罠になりやすい部分があれば特に詳しく教えてください。
```
